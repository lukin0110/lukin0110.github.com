Spark is a CDN (content distribution network) for Twitter Bootstrap, Bootswatch, Font Awsesome, some popular open-source JavaScript libraries and many more. Spark uses S3 and CloudFront to deliver the shizzle.

Advanced

Spark can be loaded from 4 different locations. Check out Rule 2 of High Performance Web Sites by Steve Souders

  • http://spark.lukin.be (default)
  • http://spark1.lukin.be (parallel)
  • http://spark2.lukin.be (parallel)
  • http://d10baajdvc8wz6.cloudfront.net (parallel)
